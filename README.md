# Cafe Delight

Simple static website showcasing a café UI built with HTML, CSS, and JavaScript.

## Project structure

- `index.html` - main page markup
- `style.css` - styling
- `script.js` - interactivity logic
- `README.md` - project overview

## Features

- Responsive layout
- Gallery/menu display
- Interactive elements

## Local setup

1. Open `index.html` in your browser.
2. (Optional) Use Live Server extension or local web server for development.
